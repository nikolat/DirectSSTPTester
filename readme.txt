■概要

  DirectSSTPを送信するシンプルなテストツールです。
  ゴーストのデバッグというよりもSSTPの送受信自体をテストすることを目的としており、
  受信したレスポンスも表示します。

■機能

  起動時にdefault.txtを取り込みます。
  よく使う送信内容を登録しておくと便利です。
  同梱したsample*.txtも送信テキストエリアにドラッグ＆ドロップすることで
  取り込むことができます。
